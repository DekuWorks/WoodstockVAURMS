Upload Guidelines

Required Columns

  • account_id - Unique customer identifier
  • bill_period - Billing period (e.g., "2024-01", "FY2024")
  • customer_class - Customer type (residential, commercial, industrial)
  • consumption - Usage amount (gallons, kWh, etc.)
  • amount - Bill amount in dollars
  • paid_flag - Payment status (true/false, 1/0, yes/no)

File Requirements

  • Maximum file size: 100MB
  • Supported formats: CSV, XLSX, XLS
  • First row should contain column headers
  • Data should be clean and consistent

Processing

After upload, files will be validated and processed. You'll receive notifications about any issues found during validation.
